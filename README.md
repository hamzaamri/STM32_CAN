The goal of this project is to establish a reliable data communication link between two STM32F407VG microcontrollers via the CAN bus. This setup can be used to simulate a range of applications, such as data exchange in automotive networks, industrial control, and sensor communication, where robustness and noise immunity are critical.

Project Objectives:
-Establish a CAN Bus Communication Network between two STM32F407VG microcontrollers.
-Transmit and Receive Data reliably between the microcontrollers, simulating real-world CAN-based communication.
-Implement Message Filtering to selectively accept messages based on CAN IDs, illustrating how nodes in a CAN network can filter out unnecessary data
